光触媒の原理

光触媒とは？

光触媒とは光が当たると触媒作用をもたらす物質のことです。

この触媒作用を用いて様々な化学反応を引き起こします。

最近ではコロナウイルスを分解し、抗ウイルス活性を示すことも話題となっています。 [1] また、クリーンエネルギーとして太陽光を用いた光触媒による水の分解による水素製造も注目を浴びています。

光触媒の原理


光触媒の原理として光触媒の水の分解を例に挙げて以下に図を示します。


図1. 水の分解における光触媒の原理


光触媒は主に半導体でこの半導体に光が照射され、光電効果により価電子帯にあった電子が伝導帯に励起して電子が存在した場所に空きができ、これが仮想のプラスの電荷として正孔が生じます。

　4H+ + 4e- → 2H2
　　 2H2O + 4h+ → O2 + 4H+
2H2O + 4e- + 4h+ → 2H2 + O2

励起した電子と正孔がそれぞれ水を酸化し、プロトンを水素に還元します。



現在の光触媒の効率



現在の光触媒の効率として神戸大学と大阪大学の研究グループが、エネルギー変換効率を7％で水の分解により水素が生成することができました。[2]

また、NEDOと人工光合成化学プロセス技術研究組合（ARPChem）は、信州大学、山口大学、東京大学、産業技術総合研究所と共同で、紫外光領域ながら世界で初めて100％に近い量子収率（光子の利用効率）で水を水素と酸素に分解する粉末状の半導体光触媒を開発しました。[3]

これらの進歩から近い将来、光触媒での水素製造の実用化が広がるかもしれません。


課題とこれからの展望


ナノ化

光触媒の試料が大きいと比表面積が小さく、反応する場所が少ないです。このことから光触媒の試料をナノ化することによって反応サイトを増やし、光触媒活性を上げる取り組みがなされています。

可視光の利用

多くの光触媒は白色で紫外線を吸収することで触媒として働きます。しかし光触媒を使用する太陽光のもとでは紫外光は3%ほどしか含まれておらず、太陽光に多く含まれる可視光の使用が望まれます。

この解決策として酸化物の光触媒を窒化させ、窒化物として光触媒を作成することがあげられます。酸素の2p軌道のバンドを窒素の2p軌道に置き換えることによりバンドが上がり、黄色の光触媒が青色の光を吸収することができます。

また、色素などを使って可視光を取り入れるアプローチも進められています。

電荷分離

光触媒に光が照射されることにより励起電子と正孔が生まれますが、この励起電子と正孔を素早く分離することが課題となっています。

助触媒は光触媒の表面に担持することにより光触媒の電荷分離を促進する役割があります。
